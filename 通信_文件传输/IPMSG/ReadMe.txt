						ReadMe
使用方法：
	# make clean; make （linux 下输入该行命令 生成可执行文件run)
	# ./run [user] [host] [picnum]
					(注：user 用户名，host 主机名，picnum 头像号码)
有待提高的地方：
	无法正确接收大文件 还不是很稳定 广播不能实现全网段（需要在代码中指定 192.168.220.255） 因为使用select实现 所以键盘接收 与收消息同步还是不太理想 可以改成线程试试
	还有好多功能没有实现（多文件传输 文件传输中断等等）
不算太无聊的添加功能（或改进）：
	1：用户首次登陆时 通过输入[picnum] （0- 100）可以设置自己的头像 （IPMSG 2.0以上有显示） 
	2：通过命令state 1 显示上线 0 隐身  可以实现类似QQ的隐身功能
	3：通过lsfile命令 可以查看当前有哪些文件是 可以下载的 通过文件前的序号 可以直接执行下载功能
	4：getfile 时 可以在窗口的第一行显示 下载进度
	5：通过lsdir命令 可以查看./run 所在目录下的文件 便于文件发送时使用
	6：（考虑到用户名重复的可能很大 并方便输入） 在命令后使用序号（命令“ls”， 唯一）代替用户名  例如：send [usernum] [msg] -> send 2 test！
	7：使用动态的结构体数组 代替结构体链表（实现简单 释放更容易 查找速度更快）
